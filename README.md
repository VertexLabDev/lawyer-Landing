# James Carter Law – Landing Page

A clean, responsive landing page I built for a law firm. Nothing fancy, just solid HTML/CSS with a contact form and a professional look.


## What's inside
- 5 sections: Hero, Practice Areas, About, Contact, Footer
- Contact form with basic validation
- Smooth scroll navigation
- SVG icons from Heroicons (inline, no extra requests)
- Mobile-friendly (tested on iPhone and Android)

## Why I built this
Wanted to have a real-world example in my portfolio. Something a service-based business could actually use. Lawyers don't need complex apps — they need something that looks trustworthy and converts visitors into calls or emails.

## Tech stack
- HTML5
- CSS3 (Flexbox for layout, nothing overkill)
- Vanilla JavaScript (just enough for the smooth scroll and form alert)
- Google Fonts (Inter)
- No frameworks — keeps it lightweight
